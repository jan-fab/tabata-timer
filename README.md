# Tabata Timer
 
A lightweight, mobile-friendly interval timer for Tabata and HIIT workouts. Configure your prepare, work, rest, cycles, sets, and cool-down phases — then hit start.
 
Hosted on GitHub Pages: **[jan-fab.github.io/tabata-timer/tabata-timer.html](https://jan-fab.github.io/tabata-timer/tabata-timer.html)**
 
## Install
 
Works as a Progressive Web App (PWA) — no app store needed.
 
- **Mobile (iOS/Android):** open in your browser, tap *Share → Add to Home Screen* to install it as an app on your phone.
- **Desktop:** open in Chrome or Edge and click the install icon in the address bar to add it to your desktop.
